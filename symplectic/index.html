qclif.SymplecticArray

Bases: DnaryArray

Class implementing symplectic algebra on matrices and vectors over , the modulo- integers. Standard array operations are inherited from DnaryArray.

See the .inner_product classmethod.

Parameters:

    The other vector.
Returns:
  • int –

    The symplectic inner product of the calling vector with the other vector.
Tip

Python's pipe operator | is overloaded with this method, so you can call v1.inner_product_with(v2) as v1 | v2.

Examples:

>>> class S3(SymplecticArray): d=3
>>> v1 = S3([1, 0, 0, 0])
>>> v2 = S3([0, 0, 1, 0])
>>> v1 | v2
2
